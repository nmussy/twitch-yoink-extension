# yoink-discord-bot

## Pre-requirements

* Node 16+
* yarn
* AWS S3 access

## Environment setup

```env
DISCORD_TOKEN=
DISCORD_CLIENT_ID=
DISCORD_GUILD_IDS=comma,separated,ids

BUCKET_NAME=
```

## Invite the bot

`https://discord.com/api/oauth2/authorize?client_id=<client_id>&permissions=0&scope=bot%20applications.commands`
